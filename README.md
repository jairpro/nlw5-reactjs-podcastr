# NLW#5 - Trilha React JS - Podcastr
Aplicação web para podcasts

### Criando o projeto:
```shell
npx react-next-app podcastr
```

### Dependências:
```shell
yarn add typescript @types/react @types/node -D
```

### Sass:
[sass-lang.com](https://sass-lang.com/)
```shell
yarn add sass
```
